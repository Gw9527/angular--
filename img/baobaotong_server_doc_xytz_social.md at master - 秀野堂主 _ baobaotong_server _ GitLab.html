1. 点赞

接口名：

    api.ext.bbw.social.gooding

说明：

    这是两向接口，点一次赞，再点一次取消赞

提交数据：

{
    token 
    uid
    goodcate //1 保屋 2 日志 3成功案例
    clickid  //日志id、保屋id 或者成功案例

}

返回数据：

{
    "status": 1,
    "msg": "点赞成功！",
    "data": {
        "clickid": "0",
        "goodcate": "1"
    },
    "timeSpend": "9.0298653ms"
}

3. 分享

接口名：

api.ext.bbw.my.share

提交数据：

{
  token 
  uid
  comefromplatform: 1001   //保宝通走1001，投保拍走1000
  mybusinessid:保屋id
  rbid:红包id
  dailyid:日志id
  sharecate:分享类型//分享是保屋、红包、日志
}

返回数据：

{
  msg: '成功',
  status: 1,
  data:
    {
    任务完成信息
  }
}
